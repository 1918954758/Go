1. Go标记	Go程序可以由多个标记组成，可以是关键字，标识符，常量，字符串，符号，如以下GO语句由6各标记组成：	fmt.Println("hello, world!")	分别是(每一行是一个标记)：	fmt	.	Println	(	"hello, world!"	)2. 行分割符	在Go程序中，一行代表一个语句结束。每个语句不需要像Java那样以;结尾，因为这些工作将由Go编译器自动完成。如果打算将多个语句写在同一行，则必须使用;认为区分，但在实际开发中，我们并不会这样做。	fmt.Println("hello, world!")	frm.Println("hello, wwwwww")3. 注释	和Java一样4.标识符	标识符用来命名变量、类型等程序实体。一个标识符实际上就是一个或多个字母(A-Z和a-z)数字(0-9)、下划线(_)组成的序列，但是第一个字符必须是字母或者下划线，而不可以使用数字开头。		以下是有效标识符：			mahesh kumar abc move_name a_123 myname50 _temp j a23b9 retVal		以下是无效表示符：			1ab (数字开头)			case (关键字)			a+b (运算符是不允许的)5. 字符串的连接	Go语言的字符串可以通过 + 实现：	package main	import "fmt"	func main(){		fmt.Println("Google" + "Runoob")	}	output=>GoogleRunoob6. 关键字	以下列举了Go代码中会使用道德25个关键字或保留字：	break, default, func, interface, select, case, defer, go, map, struct, chan, else, goto, package, switch, const, fallthrough, if, range, type, continue, for, import, return, var	除了以上介绍的关键字，Go语言还有36个预定义标识符：	append, bool, byte, cap, close, complex, complex64, complex128, uint16, copy, false, float32, float64, imag, int, int8, int16, uint32, int32, int64, iota, len, make, new, nil, panic, uint64, print, println, real, recover, string, true, uint, uint8, uintptr7. Go语言的空格	Go语言中变量的声明必须使用空格隔开，如：	var age int;	语句中适当使用空格能让程序更容易阅读。	fruit=apples+oranges;	fruit = apples + oranges;
